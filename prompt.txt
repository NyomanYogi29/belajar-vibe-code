Isi dari perencanaannya sebagai berikut:

Enum:

role: MURID | GURU | ADMIN

1. Buat tabel users:
id uuid primary key
name varchar(255) not null
email varchar(255) not null unique
password varchar(255) not null
tryoutAttempt int default 0 not null
totalPoint int default 0 not null
coinBalance int default 0 not null
role enum('MURID', 'GURU', 'ADMIN') default 'MURID' not null
isActive boolean default true not null
blockedUntil timestamp null
deletedAt timestamp null
createdAt timestamp default current_timestamp not null
updatedAt timestamp default current_timestamp not null
avatar text null
unique(email, deletedAt) // untuk user yang menghapus email nya, dan ingin kembali menggunakan email tersebut di kemudian hari
